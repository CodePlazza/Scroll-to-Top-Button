Scroll to Top Button

This example demonstrates how to create a "scroll to top button" that becomes visible when the user starts to scroll the page.

Scroll down to the page and the button will be visible.

jQuery Script

If your page is not loading jQuery - Insert the below code before the body tags or to the head section

<script src="https://code.jquery.com/jquery-3.6.1.min.js" </script>

Insert the below code before the body tags or to the head section

<script> $(document).ready(function(){ // When scrolled this far, the button appears. var offset = 20; // Button appears / disappears and page scrolls up in that milliseconds time. var duration = 300; $(window).scroll(function() { if ($(this).scrollTop() > offset) { $('#scrolltotopbutton').fadeIn(duration); } else { $('#scrolltotopbutton').fadeOut(duration); } }); $('#scrolltotopbutton').click(function(event) { event.preventDefault(); $('html, body').animate({scrollTop: 0}, duration); return false; }) }); </script>

HTML

Insert the below code before the body tags

<div id="scrolltotopbutton" style="display: none;"> <span>Top</span> </div>

STYLE

Insert the below code before the body tags

<div id="scrolltotopbutton" style="display: none;"> <span>Top</span> </div>
